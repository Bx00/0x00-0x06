0x00-0x06 - Invisibles

0x00, 0x01, 0x02, 0x03, 0x04, 0x05 and 0x06 are the most simple of cyber-weapons. They are bundled together under the name 0x00-0x06 - Invisibles.

0x00-xmas-packager

A simple program unit that invisibly hides computer files inside of digital videos. The videos can be shared on social networking platforms, such as YouTube. With a machine-learning approach, hidden data can also be detected with some accuracy.

0x01-crystal-cloak

An experimental program unit meant to create realistic images forging a person's identity; by rendering a false person in their place.

0x02-dont-be-greedy

A simple program unit that adds an invisible, resilent digital signature to C++ source code. The signature doesn't affect the operation of the code and is highly difficult to remove. This guarantees the authenticity of the author.

0x03-crystal-dagger

A simple program unit that invisibly performs digital data hiding within Persian/Urdu/Arabic text.

0x04-haha-idiot

A trivial program unit that hides invisible chatroom dialogue while fooling the observer with subtle English mistakes.

0x05-granularity

A simple program that silently creates two pages in every HTML file. One page has the regular content and is clearly visible with the browser's "View Source". A second, invisible stegano page contains the true web content. Anyone sees the false page, but a key is needed to view the true page. This can be used to host illicit websites.

0x06-gooby-pls

Checks digital images to see if they're likely photoshopped.
